ABOUT

This website was created by Fernando Munoz for the 2021 Coding and Programming Event. This website was created on angular. The event guidelines consisted of creating a stand alone program that generated the reesults of a five question quiz.

The project contains two repositories both available on github. One repsitory contains all the original code that was used to create this website. The second respository contains all the code in the format needed in order to host the app from github pages.
